# 🛒 MERN E-Commerce Website

A modern, full-featured E-Commerce web application built using the MERN Stack with secure authentication, product management, order handling, and a clean Tailwind-powered UI.

This project is divided into Frontend and Backend directories for clean separation of logic.

🚀 Tech Stack
Frontend

⚛️ React.js 🎨 TailwindCSS 🧭 React Router 📦 npm ✨ ESLint 

Backend

🟢 Node.js 🚀 Express.js 🍃 MongoDB 🧬 Mongoose 💥 Nodemon 🔐 JWT Authentication 💲Razorpay

🌱 Environment Variables

Create a .env file inside backend/:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
JWT_EXPIRE=7d
PORT=5000
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret


🧠 Requirements

Make sure you have installed:

Node.js

MongoDB or MongoDB Atlas

npm

▶️ How to Run the Project
1️⃣ Clone the repository
git clone https://github.com/Aryu-dev-journey/E-Commerce.git
cd E-Commerce

🎨 Running the Frontend
cd ./frontend
npm install
npm run dev

🔧 Running the Backend
cd ./backend
npm install
npm run dev
